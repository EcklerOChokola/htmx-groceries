# Grocery list htmx demo

Simple app to display and edit a grocery list 

## Tech

Uses htmx, runs on a simple SpringBoot app

## Run

Simply run with maven : 

```shell
mvn spring-boot:run
```

Build and run :

```shell
mvn package
java -jar ./target/htmx-table-demo-0.0.1-SNAPSHOT.jar
```

In both cases, the app is then run on the HTTP port 8080
